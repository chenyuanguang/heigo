# 安装

### 直接下载 / CDN 引用

[https://unpkg.com/heigo](https://unpkg.com/heigo)


``` html
<script src="/path/to/heigo.js"></script>
```

### NPM

``` bash

npm install heigo --save

```

### Yarn

``` bash
yarn add heigo
```

在一个模块化的打包系统中:

``` js
import $ from 'heigo'
```
